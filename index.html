Get started - Vagrantfile

If you are developing a Wordpress theme you can start with this example Vagrantfile:

VAGRANTFILE_API_VERSION = "2"

Vagrant.configure(VAGRANTFILE_API_VERSION) do |config|

  # using prepared `wpbox` box which comes with installed apache, php-fpm, mysql and wordpress
  # wordpress admin credentials - username: admin, password: admin
  config.vm.box = "wpbox"
  config.vm.box_url = "https://github.com/michaloo/wpbox/releases/download/v0.0.1-alpha/wpbox.box"

  # expose ip of the VM (wordpress is installed to `www.wordpress.local` domain)
  config.vm.network :private_network, ip: "192.168.33.10"

  # make memory limit of the VM higher
  config.vm.provider :virtualbox do |vb|
    vb.customize ["modifyvm", :id, "--memory", "1024"]
  end

  # mount source of our theme into wordpress installation (`/var/www` directory)
  config.vm.synced_folder "src/", "/var/www/wp-content/themes/example"

  # activate it (to prevent clicking in wordpress admin panel)
  config.vm.provision "shell", inline: "cd /var/www && wp theme activate example", privileged: false

end

After issuing vagrant up command and setting www.wordpress.local pointing to 192.168.33.10 ip address, you can start developing your theme!
